# pacman
This is a console-based Pacman Game developed in Java using Object-Oriented Programming (OOP) principles. Players control Pacman through a maze, collect regular and special food, avoid ghosts, and aim to clear the maze before losing all lives. The game also records scores and game history using file handling.

✨ Features
🎮 Console-based Pacman gameplay
🗺️ Fixed 20 × 30 maze with walls and pathways
👻 Four randomly moving ghosts
🍎 Regular food worth 1 point
⭐ Special food worth 5 points
❤️ Three player lives
📊 Real-time score and remaining food display
💾 Automatic game history saving
📜 View previous game results
🔄 Play Again option
